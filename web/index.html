DartCrypto

Designed and coded by Ilya Bardinov

Follow @ilya_bardinov on Twitter

DartCrypto is an OpenSource web application under the Apache License where you can convert, encode and decode content between different format systems. This happens fully in your browser using JavaScript, compiled from Dart, no content will be sent to any kind of server. Any feedback appreciated, just leave me a tweet or contribute.

Attention! Some encryption methods offered below are very basic and therefore not considered as secure.

  • Encoding
    • Encodings
  • Block Ciphers
    • Hill Cipher
    • AES (Rijndael)
    • GOST 28147-89 (Magma)
  • Stream Ciphers
    • Caesar Cipher
    • Affine Cipher
    • Vigenere Cipher
    • Beaufort Cipher
    • One-time Pad (OTP)
  • Public Key Ciphers
    • RSA PEM Generator
    • RSA
  • Hashes
    • SHA-1
